class ELDAmwl.extinction.operation.CalcExtinction

Bases: BaseOperationFactory

creates a class for the calculation of an extinction coefficient

Returns an instance of BaseOperation which calculates the particle extinction coefficient from a Raman signal. In this case, it will be always an instance of CalcExtinctionDefault().

Keyword Arguments

  • ext_params (ELDAmwl.extinction.params.ExtinctionParams) – retrieval parameter of the extinction product

  • slope_routine (ELDAmwl.bases.factory.BaseOperation) – result of ELDAmwl.extinction.tools.operation.SignalSlope

  • slope_to_ext_routine (ELDAmwl.bases.factory.BaseOperation) – result of ELDAmwl.extinction.tools.operation.SlopeToExtinction

  • raman_signal (ELDAmwl.signals.Signals) – Raman signal

  • empty_ext (ELDAmwl.extinction.product.Extinctions) – instance of Extinctions which has all meta data but profile data are empty arrays
