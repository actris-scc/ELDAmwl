class ELDAmwl.bases.factory.BaseOperationFactory¶

Bases: object

Base class of operations.

Base class of operations, returns an instance of a BaseOperation. If several alternative BaseOperation classes are available, the factory decides, which one to provide. This decision is based on options in the database or whether user defined plugins are available.

If arguments or keywords are provided, they are automatically passed to the BaseOperation instance.
