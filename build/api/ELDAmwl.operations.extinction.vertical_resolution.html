ELDAmwl.operations.extinction.vertical_resolution.operation module¶

Classes:

ExtEffBinRes()

Creates a Class for the calculation of the effective bin resolution for a given number of bins used in the retrieval of the signal slope.

ExtUsedBinRes()

Creates a Class for the calculation of how many bins have to be used for the linear fit in order to achieve the required effective bin resolutionof signal slope.

LinFitEffBinRes(**kwargs)

The calculation is done according to Mattis et al. 2016 with the equation eff_binres = round(used_bins * 0.85934 - 0.17802).

LinFitUsedBinRes(**kwargs)

The calculation is done according to Mattis et al. 2016 with the equation used_bins = round((eff_binres + 0.17802) / 0.85934).
